About DOME

Welcome to DOME: the Data Operations Methodology Explorer. DOME represents a community-driven effort, originating within the ELIXIR Machine Learning Focus Group, to improve the transparency, reproducibility, and reusability of machine learning methods applied in the life sciences and beyond. It comprises two key components: the DOME Recommendations and the DOME Registry.

DOME Registry

Implementation

The DOME Registry is the practical, web-based platform designed to implement the DOME Recommendations. It serves as a public, searchable database for detailed ML method descriptions.

Functionality

Researchers can submit their method descriptions, structured according to the DOME Recommendations (often via the DOME Wizard tool), into the Registry. Each entry receives a persistent, unique identifier. The Registry allows users to submit, discover, access and cite.

Purpose

To act as a central hub for FAIR ML method descriptions, complementing traditional publications by providing the necessary granular detail that papers often cannot accommodate. It directly supports transparency and reproducibility efforts.

Integration

The Registry uses ORCID for submitter authentication and attribution, facilitates potential accreditation via APICURON, and aims to integrate with journal publishing workflows.

Ensuring Resilience

To ensure the long-term availability and resilience of the valuable data within the DOME Registry, we aim to establish mirroring mechanisms.

Benefits

This approach safeguards against single-point-of-failure scenarios, improves access speed for a global audience, and helps guarantee the persistence of the collected method descriptions.

Concept

Mirroring involves maintaining synchronized copies of the Registry's database and potentially its interface at geographically distributed locations, often leveraging infrastructure provided by partners like ELIXIR Nodes.

Status

Mirroring infrastructure is currently under development.
