Optimization score distribution

Optimization (training) adjusts model parameters, hyperparameters, and preprocessing steps to maximize performance. Poor optimization can cause overfitting (excellent training performance but poor generalization) or underfitting (too simple models for complex data). Feature selection can help reduce overfitting but must be applied only to training data to avoid inflated performance estimates.
